# Lodge Landing Pages

Deux landing pages complètes pour les hébergements Blue Lodge et Green Lodge.

## Technologies

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- React Router
- Vite

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Déploiement sur GitHub Pages

```bash
npm run deploy
```
