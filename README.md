# Lodge Landing Pages

Deux landing pages complètes pour les hébergements Blue Lodge et Green Lodge.

## Technologies

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- React Router
- Vite

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Déploiement sur GitHub Pages

```bash
npm run deploy
```

Le site sera disponible à l'adresse : `https://[votre-username].github.io/lodge/`
